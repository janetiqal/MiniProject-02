My CSS Cheatsheet

Border Radius

Use this to curve edges of boxes. 

  
.roundbutton { 
  padding:10px;
  border-radius:10px;
    }

Float Property

The float property is positioning and formatting content.

 
img {
  float: right;
    }

Text Overflow Ellipsis Property

This puts an ellipsis after text when it does not fit in a box.

 
.overflow {
  text-overflow: ellipsis;
    }

Opacity and Transparency

The opacity property specifies the opacity/transparency of an element. It can take a value from 0.0 - 1.0. The lower value, the more transparent.


div {
  background-color: blue;
  opacity: 0.3;
    }

Z-Index

This specifies the stack order of an element on a page.

 
img{
  position: fixed;
  left: 0;
  top: 0;
  z-index: 1;
    }

The CSS Box Model

The CSS box model is basically a box that wraps around all html elements. It consists of Margin, Border, Padding, and the content.

    div {
  width: 300px;
  border: 15px solid 
  green;
  padding: 50px;
  margin: 20px;
    }
